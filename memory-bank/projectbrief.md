# Краткое описание проекта Apix (Slot Manager)

**Название проекта:** Apix (Slot Manager)

**Цель:** Заменить существующее решение для учёта работы игровых аппаратов типа «хватайка». Обеспечить автономный ввод обслуживаний (включая фото) при отсутствии интернета, фоновую синхронизацию после восстановления связи, аналитические отчёты и полный аудит изменений.

**Пользователи:**
- **Техник** (до 15 человек): ввод обслуживаний, работа оффлайн, просмотр своих аппаратов.
- **Администратор** (до 2 человек): управление справочниками, отчёты, аудит, управление техниками.

**Ключевые ограничения MVP:**
- Нет редактирования уже синхронизированных обслуживаний (только удаление через админку).
- Отчёты только онлайн.
- Техник не может удалять/редактировать синхронизированные записи.
- iOS: Background Sync недоступен, синхронизация только при открытом приложении.
- Импорт исторических данных отложен.
