# Контекст продукта — Apix (Slot Manager)

## Какую проблему решает
Существующее решение не обеспечивает надёжную оффлайн-работу и прозрачную аналитику. Проект заменяет его на современное PWA.

## Как система работает с точки зрения пользователя

### Техник
- Открывает приложение, видит список закреплённых аппаратов.
- Может работать без интернета — все справочники и последние 5 обслуживаний уже в кэше.
- Сканирует QR-код или ищет аппарат по номеру/адресу.
- Заполняет форму обслуживания: счётчики, игрушки, 3 фото, статус аппарата.
- Видит предварительный расчёт (если онлайн) и тренд ROI.
- Сохраняет черновик в IndexedDB.
- При появлении интернета синхронизирует черновики с сервером.
- Получает ненавязчивый баннер, если есть «забытые» аппараты (превышен интервал обслуживания).

### Администратор
- Видит дашборд всех аппаратов с выручкой, ROI и трендами.
- Управляет справочниками (локации, маршруты, игрушки, типы аппаратов, сотрудники).
- Просматривает журнал обслуживаний, может удалить запись (с пересчётом цепочки).
- Формирует финансовые отчёты, экспортирует в Excel.
- Выполняет замену аппарата на точке (закрытие старого placement, создание нового).
- Просматривает аудит всех изменений.
